Reverse1999-Character-Manager
🌟 项目简介
这是一个大一学生的第一个Java练习项目，为《重返未来：1999》玩家设计的角色信息管理工具，帮助整理和查询游戏角色数据。 从零开始构建一个完整的桌面应用程序，涵盖从数据建模到界面设计的全流程。使用JavaFX开发，支持JSON数据存储。
