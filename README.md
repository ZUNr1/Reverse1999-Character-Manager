

##🌟 项目简介##

这是一个大一学生的第一个Java练习项目，一个灵感来源于《重返未来：1999》的粉丝向角色创建与查询平台
从零开始构建一个完整的桌面应用程序，涵盖从数据建模到界面设计的全流程。
使用JavaFX开发，支持JSON数据存储。
该项目为非官方粉丝作品，与游戏官方无关，仅用于个人学习目的，不会生成release版本

##🎯 已完成功能##

📚 角色数据中心 - 完整的角色数据模型设计与管理
👥 角色管理 - 支持角色的录入、编辑与详细信息查询
💾 数据持久化 - 基于JSON的本地数据存储方案
🎨 图形化界面 - 使用JavaFX构建的用户友好界面

##🔄 开发计划##

🗄️ 数据库升级 - 从JSON迁移至MySQL，实现更强大的数据管理
🔍 智能筛选 - 多条件组合查询与高级搜索功能
🌐 Web化改造 - 向Spring Boot + 前端技术栈演进

##📅 未来规划##

📊 数据分析 - 角色属性统计与可视化展示
🎴 技能系统 - 完整的技能管理与搭配推荐
📖 游戏术语库 - 专有名词解释与查询图鉴

##🎯 核心功能##

🔍 多条件筛选
📚 专有名词查询图鉴（游戏术语解释与查询）
📊 角色属性管理

Maven 3.6+

架构设计：MVC模式 + 建造者模式

代码质量：完整的输入验证和异常处理
